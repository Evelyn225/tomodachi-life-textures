How to use textures in Tomodachi Life: Living the Dream

Using TomoTexture to swap drawings in your save file
What is TomoTexture?

TomoTexture is a free tool by AlfonsoMallozzi that lets you replace the user-generated content (UGC) drawings in your Tomodachi Life: Living the Dream save file with any image you choose (including textures downloaded from this site).

AlfonsoMallozzi/TomoTexture - Download latest release
What can be replaced?
Face Paint Food Treasures (most goods)

Videos cannot be replaced. Each texture type has specific required dimensions. The textures on this site are already the correct size for their category.

How to replace a texture
1
Download TomoTexture
Grab the latest release from the GitHub link above and run it.
2
Download a texture from this site
Hit the Download button on any texture. It saves as a PNG to your downloads folder.
3
Select your save folder
Point TomoTexture at the folder where your game save slots are stored. You'll also pick a backup location. TomoTexture saves a copy of your original before making any changes.
4
Choose what to replace
Browse the in-game drawings listed by TomoTexture, pick the one you want to swap out, then select your downloaded PNG.
5
Load the game
Your new texture appears in-game. If something looks wrong, TomoTexture can revert to your backed-up original.
Want to share your own textures?

If you've made a texture you're happy with, create an account and upload it so others can use it too.
